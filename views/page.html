slyncr

synchronize your slides with slyncr! view a demo

1

get the bookmarklet → slyncr

and activate it on the slideshow page

2

get the remote

enter the url of the slideshow to get the remote control

support

slyncr works with these sites:

  • scribd (html version)
  • 280slides (buggy)
  • any site that uses arrow keys for controls, like Google Docs
  • more support coming soon...

and has been tested with theses browsers:

  • Chrome 7
  • Firefox 3.6
  • probably works in more browsers, but this will be verified

built with node.js, socket.io, and express by @chrisjaure
